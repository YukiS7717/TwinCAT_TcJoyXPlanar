# TwinCAT_TcJoyXPlanar
TcJoyを使ってXBOXコントローラーでXPlanarを動作する

## 履歴

ver1.0: 新規作成 

## ボタン配置

ボタン |          内容
------ | -----------------------
Start  | Enable/Disable
Back   | Reset
Pad    | C回転可能位置へのXY移動
LStick | XY移動
RStick | AB移動
A      | Z移動
L1     | C90 CCW
R1     | C90 CW
L2     | C CCW
R2     | C CW

## 必要プログラム

[TcJoy](https://github.com/evanmj/TcJoy)
